# 吃什么 - 食材随机选择器

今天想吃些什么呢？让"吃什么"帮你解决选择困难症！

## 🎯 项目简介

这是一个简单有趣的食材/菜品随机选择器，从海量美食中为你随机推荐一款美味。支持按菜系、用餐时段、食材类型、餐品类型等多维度筛选。

## ✨ 功能特性

- 🎲 **随机选择**：一键随机选择今日美食
- 🔍 **多维筛选**：支持菜系、时段、食材类型、餐品类型筛选
- 📱 **响应式设计**：完美适配手机和电脑
- 🌍 **多平台支持**：Web版 + 微信小程序版
- ⚡ **快速部署**：支持 Netlify、Cloudflare Pages 等平台

## 🚀 快速开始

### 本地运行

直接打开 `index.html` 文件即可在浏览器中运行。

### 在线部署

详见 [部署教程.md](部署教程.md)，支持以下平台：

1. **Netlify**（推荐）- 拖拽即可部署，5分钟搞定
2. **Cloudflare Pages** - 速度快，适合长期使用
3. **Gitee Pages** - 国内访问最快

## 📊 数据说明

项目包含丰富的菜品数据（`ingredients.json`），涵盖：
- 8大菜系：川菜、粤菜、鲁菜、苏菜等
- 多种用餐时段：早餐、午餐、晚餐、夜宵
- 丰富食材类型：肉类、蔬菜、海鲜、面食等
- 多样餐品类型：主食、汤品、小吃、饮品等
