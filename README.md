# 有什么好吃的？- 智能食物推荐器

今天想吃些什么呢？让"有什么好吃的？"帮你解决选择困难症！

## 🎯 项目简介

这是一个简单有趣的食物随机推荐器，从海量美食中为你智能推荐一款美味。支持按菜系、用餐时段、食材类型、餐品类型、健康标签、饮食禁忌等多维度筛选。

## ✨ 功能特性

- 🎲 **智能推荐**：一键随机推荐今日美食，带滚动动画效果
- 🔍 **多维筛选**：支持菜系、时段、食材类型、餐品类型筛选
- 🔎 **实时搜索**：支持菜品名称搜索，快速定位（支持Enter键）
- 📚 **食物大全**：浏览所有菜品，按拼音排序
- 🚀 **双模式**：常见模式（2714道，首屏快速加载）+ 完整模式（3338道，后台懒加载）
- 📱 **响应式设计**：完美适配手机和电脑
- 🌐 **纯静态网站**：无需服务器，完全离线可用

## 📊 数据说明

项目包含丰富的食物数据，两个数据文件分工明确：

- **ingredients_common.json**：常见菜品（2714道），可在小区门口、商业街、公司旁广场购买到的日常菜品
- **ingredients_full.json**：全量菜品（3338道），包含高端餐厅、异国特色、宴席菜品等所有菜品
