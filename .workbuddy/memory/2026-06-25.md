## 任务一：三段式推荐范围切换功能

### 任务描述
用户要求在页面头部"有什么好吃的?"大标题右侧添加一个三段式循环切换按钮组件，控制"开始推荐"功能的筛选范围，并动态切换标题文本。

### 功能规格

三种推荐模式（从左至右）：

| 模式 | 按钮文本 | 标题文本 | 推荐范围 |
|------|----------|----------|----------|
| `food`（默认） | 好吃的 | 有什么好吃的? | 仅食物类，剔除酒水、饮品 |
| `all` | 全部 | 有什么推荐的? | 全部内容类型 |
| `snack` | 小吃零食甜点饮品 | 有什么解馋的? | 仅小吃、零食、甜点、饮品 |

## 任务二：功能独立性修正

### 任务描述
修正"开始推荐"功能的严重逻辑缺陷：推荐基于上一条结果进行相似推荐，而非纯随机。同时修正搜索和食物大全受常见/全部筛选影响的问题。

### 问题分析

| 问题 | 原因 | 影响 |
|------|------|------|
| "开始推荐"基于历史推荐相似推荐 | `handleSelect()` 使用 `pickRelatedItem(availableItems, state.lastResult)` | 推荐结果不独立，受上一条影响 |
| 搜索受常见/全部筛选影响 | `updateFilteredItems()` 中包含 `showAllMode` 过滤 | "常见"模式下搜索范围受限 |
| 食物大全可能显示 common 子集 | `handleCatalog()` 使用 `state.ingredientsData.items` | 懒加载完成前目录不完整 |
| 搜索可能使用 common 子集 | `showSearchDropdown()` 使用 `state.ingredientsData.items` | 懒加载完成前搜索范围受限 |
| 统计总数可能为 common 数量 | `updateStats()` 使用 `state.ingredientsData` | 显示不完整总数 |
| 懒加载仅在 `showAllMode` 时更新 | `lazyLoadFullData()` 有 `if (state.showAllMode)` 条件 | "常见"模式下数据不刷新 |

### 修正方案

#### 1. 添加 `state.fullData`
- 新增 `state.fullData` 始终持有 full.json 完整数据
- 供搜索、食物大全、统计总数使用

#### 2. 分离常见/全部过滤
- 从 `updateFilteredItems()` 移除常见/全部过滤（`showAllMode`）
- 将常见/全部过滤移至 `getRecommendationItems()`
- `state.filteredItems` 现在仅包含搜索词 + 用户筛选条件，不含常见/全部

#### 3. "开始推荐"改为纯随机
- `handleSelect()` 移除 `pickRelatedItem()` 和 `state.lastResult` 逻辑
- 改为 `Math.floor(Math.random() * recommendItems.length)` 纯随机选取

#### 4. 搜索和食物大全使用 fullData
- `showSearchDropdown()` → `state.fullData?.items || state.ingredientsData.items`
- `selectFromSearchDropdown()` → 同上
- `handleCatalog()` → 同上

#### 5. 统计使用 fullData
- `updateStats()` 总数使用 `state.fullData?.items?.length`

#### 6. 懒加载始终更新
- `lazyLoadFullData()` 移除 `if (state.showAllMode)` 条件
- 始终设置 `state.fullData` 并调用 `updateFilteredItems()` + `updateStats()`

#### 7. 简化 handleToggleCommon
- 移除 `updateFilteredItems()` 和 `updateStats()` 调用（不再需要）
- 仅保留 UI 状态切换和 URL 更新

### 数据流架构（修正后）

```
full.json → state.fullData → state.filteredItems → getRecommendationItems()
                │                  │                       │
                ├→ 搜索             ├→ 统计数字             ├→ 开始推荐（纯随机）
                ├→ 食物大全         └→ （不含常见/全部）     └→ 大家还喜欢（相似推荐）
                └→ 统计总数
```

| "大家还喜欢"相似推荐（藕饼→水煮蛋/松仁小肚/煎蛋/烩面） | ✅ 通过 |
