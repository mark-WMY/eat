# 有什么好吃的？- 智能食物推荐器

今天想吃些什么呢？让“有什么好吃的？”帮你解决选择困难症！

## 🎯 项目简介

这是一个简单有趣的食物随机推荐器，从海量美食中为你智能推荐一款美味。支持按菜系、用餐时段、食材类型、餐品类型、健康标签、饮食禁忌等多维度筛选。

## ✨ 功能特性

- 🎲 **智能推荐**：一键随机推荐今日美食，带滚动动画效果
- 🔍 **多维筛选**：支持菜系、时段、食材类型、餐品类型筛选
- 🔎 **实时搜索**：支持菜品名称搜索，快速定位（支持Enter键）
- 📚 **食物大全**：浏览所有菜品，按拼音排序
- 📱 **响应式设计**：完美适配手机和电脑
- 🌐 **纯静态网站**：无需服务器，完全离线可用

## 🚀 快速开始

### 本地运行

直接打开 `index.html` 文件即可在浏览器中运行。


## 📊 数据说明

项目包含丰富的食物数据（`ingredients.json`），涵盖3339条国内外常见菜品：
- **八大菜系**：川菜、粤菜、鲁菜、苏菜、浙菜、闽菜、湘菜、徽菜等
- **国际美食**：日料、韩料、西餐、东南亚菜等
- **用餐时段**：早餐、午餐、晚餐、夜宵、下午茶
- **食材类型**：肉类、蔬菜、海鲜、面食、豆制品等
- **餐品类型**：主食、汤品、小吃、饮品、甜点等

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript（无框架依赖）
- **部署**：纯静态网站托管，无需后端服务器
- **数据存储**：JSON格式，完全本地化

## 📝 已实现功能

- ✅ 滚动选择动画效果
- ✅ 实时搜索功能（支持防抖和Enter键）
- ✅ 食物大全浏览（按拼音排序）
- ✅ 多维度筛选（菜系/时段/食材/餐品类型）
- ✅ 完全离线化（无外部CDN依赖）
